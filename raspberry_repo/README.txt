app-release.apk - это файл установки для Android
codeOverwiew

MainActivity - класс активности, которую пользователь видит при запуске.
Он отвечает за создание и поддержку соединения, а также отправку/получение сообщений.
MessagesAdapter - класс адаптер для RecyclerView (вид актиновсти для "бесконеночго контента")
Message - простой класс/оболочка для сообщений

Пожалуйста, если вы не увидели сообщения об успешном соединении при запуске приложения - 
1) скачайте server.py
2) запустисте его в локальной сети, к которой подлючен телефон
3) установаите новый ip командой "ip set>123.456.78.90" (где 123.456.78.90 - адресс вашего компьютера в локальной сети)
P.S. команду нужно писать прямо в приложении
